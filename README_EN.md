This project implements a **guided system** that helps users select the **most appropriate data structure** for a given problem, based on a set of 8 key questions.  

The system is built using **Python + Streamlit**, providing a simple, interactive interface that runs on any machine.

This project fulfills all official requirements:

- Questionnaire (8 key questions)
- Deterministic decision logic
- Main recommended data structure
- Explanation and rationale
- ASCII visualization of the structure
- Pseudocode of the main operations
- Example based on coursework activities
- Single-command execution system
